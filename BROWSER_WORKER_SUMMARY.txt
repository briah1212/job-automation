ARCHITECTURE
================================================================================

BrowserWorker
├── process_application()          # Main workflow
│   ├── Navigate to URL
│   ├── Detect ATS adapter
│   ├── For each form page:
│   │   ├── Inspect form fields
│   │   ├── Map fields to canonical names
│   │   ├── Fill fields
│   │   ├── Upload resume
│   │   ├── Create checkpoint
│   │   └── Navigate to next page
│   ├── Create "ready_to_submit" checkpoint
│   └── If assisted_mode: PAUSE (return awaiting_approval)
│
└── resume_from_checkpoint()       # Resume & submit
    ├── Load checkpoint
    ├── Navigate to checkpoint URL
    ├── Submit form
    ├── Detect confirmation page
    └── Return result

ATSAdapter (base class)
├── detect()                       # Can this adapter handle this page?
├── inspect_form()                 # Extract form schema
├── fill_field()                   # Fill a single field
├── upload_document()              # Upload file
├── navigate_next()                # Click next/continue
├── submit()                       # Final submit
└── detect_confirmation()          # Is this a confirmation page?

Adapters:
├── MockATSAdapter                 # For mock ATS (data-ats="mock")

Services:
├── FormInspector                  # Extract fields with labels, types, options

└── CheckpointManager              # Save/restore state with screenshots
